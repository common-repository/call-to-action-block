This plugin brings a Call to Action Block to the new WordPress Gutenberg editor.

== Description ==

Easily add a Call to Action Block to your WordPress post or page via the new Gutenberg Editor.

- Add text
- Add Sub Text
- Add Button Text
- Add Button Url
- Add Background Image
- Add Background color
- Add Text color
- Add Font Size


== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Testimonials Block"
3. **Install** the "Testimonials  Block" plugin
4. **Activate** "Testimonials Block" from your Plugins page
5. **Insert Block** on your Gutenberg Editor and select "Testimonials"
